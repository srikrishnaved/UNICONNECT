# UniConnect — ChristConnect

UniConnect is a student engagement platform built for CHRIST (Deemed to be University), Bangalore. It centralises the academic and co-curricular life of the campus into a single app — giving students a live timetable, a study planner, club discovery and event tracking, and peer-to-peer networking, while giving faculty coordinators and club administrators the tools they need to manage events, approvals, and documentation without paper.

## Features

- **Timetable Management** — real-time class schedules with substitute and cancellation tracking, faculty availability, and permanent overrides
- **Club Events** — club dashboards for admins to post events, track progress, and manage registrations; students can discover and follow clubs across campus
- **AI Study Planner** — personalised study schedules generated by Claude AI based on subjects, exam dates, and available hours
- **Documentation Generation** — one-click generation of formal NFA (Note for Approval) and Activity Report `.docx` files, with AI-assisted content generation from a poster or description
- **CR Dashboard** — class representative tools for attendance tracking, compensatory request management, and class roll submission
- **Student Networking** — profiles, club memberships, and a hub for campus-wide announcements and events

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React Native + Expo (web-first) |
| Backend & Database | Supabase (PostgreSQL + Row Level Security) |
| Edge Functions | Deno (Supabase) |
| Document Generation | docx (npm:docx@8) |
| AI | Anthropic Claude (claude-sonnet-4-6) |
